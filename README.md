# DenoDemo - Using Deno & NPM

Run a simple web server using Deno and NPM packages.

## Prerequisites

Have Deno 1.32.3 installed.

## Usage

```bash
deno run --allow-net --allow-read main.ts
```

## Browser

Open a browser to http://localhost:6969
